Tag Support

Friday, May 30, 2014 by Jeffrey Randow
The theme now supports tags, tag pages, and a tag cloud. This functionality requires you to run a utility before pushing your updates to Github Pages. In order to use tags, you need to add the following information into your YAML front matter: {% highlight yaml linenos %} tags: - Tag Name (can be multiple words) {% endhighlight %} The tag pages will be stored in the /tags directory and the tagcloud.
